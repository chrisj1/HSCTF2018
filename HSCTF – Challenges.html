  • Special Alien Print

    Aliens have come and set foot on Earth!

    (But don't worry, they just want to communicate).
    The aliens send a message to a programmer, Keith, who just happened to be nearby. Keith is interested in the aliens and she wants to be able to communicate with them.
    The aliens handed Keith some code from their language, after looking over it a few times, Keith notices a few important features.
    - Any ASCII characters that need to be printed are done by writing %$X^$, where X is the text to be printed
    - X has to be formatted properly as well. An ASCII character can be contained in X, and you write it by putting (num) inside of X, where num is the ASCII number. But there is a catch: The order of ASCII has been reversed!
    So ASCII now starts with 'DEL', then '~', and so on. For example, to print ~, X would be (1)
    - You can print multiple ASCII characters in X, like (num)(num2)
    - Finally, you can print multiple characters at a time by doing *rep after a character. For example (num)*2(num2) in place of X would print out the backwards ASCII character of num two times and then it will print num2    You need to write a program in their language to greet the aliens but there is one problem: The aliens don't have much space on their "computer." It is very tiny and primitive compared to ours. You need to be as efficient as possible to write the words "Hello, how've you been?" using their program. The flag is the program.
    NOTE: Do not print the "".

  • The Game
    Matthew Ye · 10 solves · algo · 300 pt

    Every week, Keith and his friend Jan play a game dubbed The Game. Contrary to the version the reader has just lost, this game takes place on a rooted, directed tree with all edges directed from parent to children. Each leaf is assigned a number from 1 to m, where m is the number of leaves. In the game, a token begins at the root and ends when it reaches a leaf - Keith and Jan take turns moving it down the tree along the edges. The result of the game is the value of the leaf the token ends on. Keith always goes first and tries to maximize the result of the game, while Jan tries to minimize the result of the game. Assuming Keith and Jan play optimally, find the result if Keith is allowed to arrange the values of the leaves in any order before the game, and the result if Jan is allowed to arrange the values of the leaves in any order before the game. The answer is given as KeithResult_JanResult. For example, if the result with Keith's leaf arrangement is 4 and the result with Jan's leaf arrangement is 3, the answer will be 4_3. You can find a description of the tree here - the first line gives the number of nodes N, and the remaining lines give edges u v which represent an edge directed from node u towards node v. The root is node 1. Please note that the file contains the node labels, not the values inside the nodes (which is exclusive to the leaves.)
